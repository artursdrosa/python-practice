# Python Practice
Here are some exercises to practice programming logic, code modularization, and validations with Python.

## 1 - Tips
This algorithm will calculate the tip amount based on the bill total and the customer's desired tip percentage. Then, it will display the total payment amount.

## 2 - CPF
This algorithm verify if your CPF is valid. The CPF must contain only 11 digits. 

## 3 - Vowels
This algorithm count the quantity of vowels in the entry text

## 4 - Long Words
This algorithm takes a text and separates words with more than 10 characters.

## 5 - Safe Password
This algorithm creates safe passwords completely random, always including at least one Upper Case, one Lower Case, one digit, and one special character.
